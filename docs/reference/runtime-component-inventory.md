# Runtime Component Inventory

Production Deployment Readiness Planning v0 inventory. Status values:

- `ready`: production-ready within current scope.
- `partial`: useful boundary exists, but production work remains.
- `not_ready`: implemented for local/integration use only.
- `future`: planned boundary only.

No component in this inventory should be read as approving production deployment.
